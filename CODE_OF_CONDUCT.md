# Code of Conduct

## Our Pledge

We aim to keep this project welcoming, respectful, and focused on creative historical interactive fiction.

## Expected Behavior

- Be respectful when discussing history, culture, and creative choices.
- Give constructive feedback with specific examples.
- Keep contributions aligned with the project's safety and content guidelines.

## Unacceptable Behavior

- Harassment, personal attacks, hate speech, or discriminatory language.
- Erotic content, extreme graphic violence, or modern political agitation in dream content.
- Publishing secrets, API keys, or private user information.

## Enforcement

Maintainers may edit, reject, close, or remove contributions that violate these guidelines.
