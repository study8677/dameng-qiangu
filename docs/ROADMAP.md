# Roadmap

## MVP

- Static GitHub Pages app.
- Six official pre-Qing historical figure dreams.
- AI-assisted local dream creation.
- Share links through compressed URL hash.
- Unit and E2E coverage for core flows.

## V1: Public Dream Plaza

- Supabase login.
- Public/private/unlisted visibility.
- Published dream records in Postgres.
- Likes, collections, basic comments.
- Report and takedown flow.

## V1.5: Creator Tooling

- Node graph editor.
- Dream quality validation report.
- Remix lineage.
- Creator profile page.
- Official weekly prompt challenges.

## V2: Deeper Gameplay

- More pre-Qing figures.
- Figure traits and passive abilities.
- Hidden endings and achievement gallery.
- Timeline or dynasty filters.
- Better mobile reading and sharing.

## V3: Platform

- Moderation dashboard.
- Revenue experiments.
- Template marketplace.
- Multi-language exports.
